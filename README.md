# Milwaukee City Call Center Data Analysis

## Overview
This analysis explores call center operations data from the City of Milwaukee to uncover insights about service request patterns, seasonal trends, and operational efficiency.

## Data Sources
- **callcenterdatahistorical.csv** - Historical call center records
- **callcenterdatacurrent.csv** - Current/recent call center records

Data sourced from [Milwaukee Open Data Portal](https://data.milwaukee.gov/)

## Analysis Notebook
- **milwaukee_call_center_analysis.ipynb** - Complete analysis including:
  - Data exploration and quality assessment
  - Yearly volume trends and 2020 anomaly investigation
  - Top request types analysis
  - Case closure and resolution time metrics
  - Seasonal patterns heatmap for workforce planning

## Key Findings
- Call center tracking system launched March 2020
- Top issues include brush pickup, cart issues, missed collections, and street lights
- Strong seasonal patterns identified for workforce planning
- April is peak month for infrastructure issues (potholes, drains)
- Summer months peak for vegetation-related requests
